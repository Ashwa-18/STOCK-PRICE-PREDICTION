# STOCK-PRICE-PREDICTION
# KNN Regression Project

This project demonstrates the implementation of both **K-Nearest Neighbors (KNN) Classification** and **KNN Regression** using Python and Scikit-learn.

## Features
- KNN Classification for predicting class labels
- KNN Regression for predicting continuous values(Close)
- Hyperparameter tuning using GridSearchCV
- Model evaluation using RMSE and R² score

## Results
- **KNN Regression**
  - RMSE ≈ 27.9
  - R² Score ≈ 0.60
